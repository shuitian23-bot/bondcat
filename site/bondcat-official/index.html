桌面边缘的打字 RPG

官网增强版适合想在其它应用里打字时也推进游戏的用户；App Store 版适合只需要窗口内游玩的 Mac 用户。

打字推进

键盘输入和鼠标点击会变成攻击节奏，小猫会推进、停步、攻击、拾取掉落，适合放在屏幕边缘长期陪伴。

装备与皮肤

开箱、穿装、换武器、换外观，近战、远程和法术有不同的出手节奏与战斗反馈。

安全分发

官网增强版使用 Developer ID 签名并通过 Apple notarize，可在 Gatekeeper 校验下安装。

两个版本怎么选

两个版本存档和玩法方向一致，差异主要是输入方式和分发渠道。

官网增强版
  • 支持全局键盘/鼠标输入触发战斗
  • 需要用户在系统设置中授权
  • 适合放在桌面边缘长期运行
  • Developer ID 签名 + Apple notarize
Mac App Store 版
  • 只在 BondCat 窗口激活时响应输入
  • 不需要 Accessibility 或 Input Monitoring
  • 适合想从 App Store 购买和更新的用户
  • 已经可在全球 App Store 分发

安装 macOS 版

  1. 点击“macOS 下载”，打开 DMG。
  2. 把 BondCat Desktop 拖到 Applications。
  3. 首次打开后，按提示在系统设置中允许 Accessibility 和 Input Monitoring。
  4. 重启 BondCat Desktop，把窗口放到桌面边缘即可开始。

安装 Windows 版

  1. 点击“Windows 下载”，得到 bondcat-Windows-x64.zip。
  2. 解压后运行 EXE 安装程序，安装完成后启动 BondCat Desktop。
  3. Windows 版使用全局键盘/鼠标监听，适合放在桌面边缘运行。
  4. 当前 Windows 包未接入代码签名证书，首次安装可能出现 SmartScreen 提示。
