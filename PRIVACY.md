# BondCat Privacy Policy

Last updated: June 29, 2026

BondCat is a lightweight desktop companion game. It does not require an account, does not include advertising, and does not collect personal data.

## Data Collection

BondCat does not collect, transmit, sell, or share personal data. Game progress and settings are stored locally on your Mac.

## Keyboard and Mouse Events

BondCat may request macOS Accessibility/Input Monitoring permission so keyboard and mouse activity can be counted as in-game attacks.

BondCat only counts input events for gameplay. It does not read typed text, record key values, capture passwords, inspect other apps, or send input data to any server.

## Network Use

BondCat does not require a network connection for gameplay. The app does not operate analytics, ads, user tracking, or cloud sync.

## Contact

For support or privacy questions, open an issue at:

https://github.com/shuitian23-bot/bondcat/issues

# BondCat 隐私政策

更新日期：2026 年 6 月 29 日

BondCat 是一款轻量桌面陪伴游戏。应用不需要账号、不包含广告，也不收集个人数据。

## 数据收集

BondCat 不收集、传输、出售或分享个人数据。游戏进度和设置仅保存在你的 Mac 本机。

## 键盘和鼠标事件

BondCat 可能会请求 macOS 辅助功能/Input Monitoring 权限，用于把键盘和鼠标操作计为游戏内攻击。

BondCat 只统计输入事件数量用于玩法。它不会读取输入文字、记录按键值、捕获密码、检查其他应用，也不会把输入数据发送到服务器。

## 网络使用

BondCat 不需要联网即可游玩。应用不包含分析、广告、用户追踪或云同步。

## 联系方式

支持或隐私相关问题可以在这里提交：

https://github.com/shuitian23-bot/bondcat/issues
